# Aurora Neo Theme

A modern, customizable Shopify theme designed for fashion and lifestyle stores.

## Features

- Responsive design
- Customizable color schemes
- Flexible layout options
- Built-in product galleries
- Mobile-optimized
- SEO friendly

## Installation

1. Download the theme files
2. Upload to your Shopify store via Online Store > Themes
3. Customize settings in the theme editor

## Customization

The theme includes extensive customization options:
- Color schemes
- Typography settings
- Layout configurations
- Product display options
